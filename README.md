### Make a build
1. ```npm ci```
2. ```npm run build```
3. Unpacked extension will be available in **public** folder
4. Go to Extension page in Chrome/Opera, enable **Developer mode**, then press **Load unpacked** and load **public** folder
5. Enjoy =)
