### Make a build
1. ```npm install```
2. ```gulp```
3. Unpacked extension will be available in **public** folder
4. Go to Extension page in Chrome/Opera, press *Developer mode*, then *Load unpacked extension...* and load **public** floder
5. Enjoy =)
